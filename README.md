# 🌟 BlogSpark – בלוג הניצוצות

## 🧭 הגדרה כללית

מאגר זה נועד לשמש כבלוג הרשמי של **AnLoMinus**, ובו יפורסמו כל השיחות, הפיתוחים, החידושים וההשראות שנולדים בתהליך הבריאה היומיומי שלנו. כל פוסט הוא ניצוץ של מחשבה, חוויה או הארה, אך מעבר לכך הוא גם מסע בזמן שבו הקורא והיוצר מתאחדים בחוויית גילוי מתמשכת.  
בכל מאמר מתגלה שכבה חדשה של השראה – רעיונות נולדים מתוך רמזים, טקסטים מתמזגים עם חזון, והכתיבה הופכת לטקס של יצירה מודעת.  
הבלוג מבקש לא רק לתעד שיחות אלא לשמר אנרגיה של התחדשות, להעניק השראה מתמשכת לכל מי שנוגע באור הזה. הוא נבנה כבית פתוח לכל הניצוצות שזורחים מהשיח האנושי־אלוקי, מקום שבו מחשבה טהורה מקבלת לבוש מילולי ונהפכת למקור השפעה.  
כך נרקם עולם שבו כל פוסט הוא שער לעולמות של תוכן, עומק ורגש — מסע בין מילים שהופכות לכלים של אור, ומזמינות את הקורא להצטרף למעשה הבריאה ולגלות את הניצוץ שבתוכו.

## 🎨 עיצוב מומלץ ל־GitHub Pages

- **Theme:** `minimal` או `jekyll-theme-cayman`
    
- **Fonts:** Poppins / Assistant / Source Sans 3
    
- **Colors:** #0ffbdf (Neon Cyan) / #ffd700 (Royal Gold)
    
- **Highlight:** ⚡Electric Glow + Gradient Text
    

---

## 🌐 הוראות הפעלה ל־GitHub Pages

1. פתח את קובץ `_config.yml` והוסף:
    
    ```yaml
    theme: jekyll-theme-cayman
    title: BlogSpark – בלוג הניצוצות
    description: ארכיון האור של AnLoMinus
    author: AnLoMinus
    markdown: kramdown
    plugins:
      - jekyll-feed
      - jekyll-seo-tag
    ```
    
2. עבור ל־Settings → Pages ובחר Branch: `main` + Folder: `/ (root)`
    
3. לאחר השמירה, אתר הבלוג יופיע בכתובת:  
    **[https://anlominus.github.io/Blog](https://anlominus.github.io/Blog)**
